27. Explain the recursive function with example
ans. A recursive function is a function that calls itself to solve a smaller version of the original problem. Recursive functions must have a base case to stop the recursion.
eg- function factorial(n) {
    if(n === 0) {
        return 1; // Base case
    } else {
        return n * factorial(n - 1); // Recursive call
    }
}
console.log(factorial(5)); // Output: 120

66. Explain the various creating of nodes with example
ans. 1. Creating an Element Node
Purpose: To create a new HTML element (like <div>, <p>).
Example:
let para = document.createElement("p");   // Create <p> element
para.innerHTML = "Hello World!";         // Add text inside
document.body.appendChild(para);         // Add it to body
2. Creating a Text Node
Purpose: To create text content for an element.
Example:
let text = document.createTextNode("This is a text node");
let div = document.createElement("div");
div.appendChild(text);                    // Add text inside div
document.body.appendChild(div);
3. Creating an Attribute Node
Purpose: To create a new attribute for an element.
Example:
let link = document.createElement("a");
let attr = document.createAttribute("href"); // Create href attribute
attr.value = "https://www.example.com";
link.setAttributeNode(attr);                 // Attach attribute to element
link.innerHTML = "Visit Example";
document.body.appendChild(link);
4. Creating a Comment Node
Purpose: To add a comment dynamically in the DOM.
Example:
let comment = document.createComment("This is a comment");
document.body.appendChild(comment);
5. Creating a Document Fragment (Temporary Node)
Purpose: To create a lightweight container to hold multiple nodes before adding them to the DOM.
Example:
let fragment = document.createDocumentFragment();
for(let i=1; i<=3; i++) {
    let li = document.createElement("li");
    li.textContent = "Item " + i;
    fragment.appendChild(li);
}
document.getElementById("myList").appendChild(fragment);

103.  Demonstrate pass by reference with an example program in PHP.
ans. Normally, PHP passes arguments by value, meaning the function works on a copy of the variable.
Pass by reference allows the function to directly modify the original variable.
Use & before the parameter in the function definition to pass by reference.
Example:
<?php

    $num += 10;
}
$value = 20;
addTen($value);  // Pass by reference
echo "Value after function call: " . $value;
?>
